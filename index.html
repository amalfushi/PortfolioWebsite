About me

Originally from Central Washington, I attended the University of Washington in pursuit of a BFA in Photography. My work often reflected my interests in technology and software. After graduation I spent several years working in the commercial art industry and spent time teaching myself some basic office automation to assist in my job.

I found myself drawn to programming more and more, so I took the time to teach myself the basics of computer science via the University of Helsinki's: MOOC- Java. Absolutely enamored with what I had learned, I took the plunge and enrolled at Coding Dojo: Seattle. Coding Dojo offers a bootcamp of contemporary technologies designed to create self-sufficient full-stack developers. Dojo instructors push students to not only learn to write code, but also to develop the necessary set of tools to be a comprehensive web developer. These tools are crucial for optimal success in my career as a software developer.

C# Uno

A Terminal-based version of Uno

As part of an afternoon Hackathon, I worked with two other teammates to put together a C# Version of Uno. The program runs entirely in terminal and we experimented with various colors in the command line to make the program easier to read.

We faced a number of challenges in working through the logic of running the game, but found ourselves with a successful interpretation of Uno. Interpreting user input and passing cards between methods and classes held us up for a bit. In further development, it would be great to allow for multiple players, more robust logic on opponents' turns, and making the game more visually appealing.
